    🎯 Purpose

    GitQuest was built to make learning Git hands-on and immediate. Instead of passively reading documentation, you type real commands into a live simulated terminal and watch the results — a commit graph, file tree, and branch structure — update in real time.

    The goal is to close the gap between knowing Git conceptually and being comfortable using it day-to-day. Whether you're a complete beginner or want to sharpen your understanding of branching, rebasing, and stashing, there's something here for you.

    ⚙️ How it works

    Everything runs entirely in your browser — no backend, no server, no account. Git commands are simulated by an in-memory state machine that tracks branches, commits, staged files, stashes, and remotes, producing realistic terminal output for over 25 git subcommands.

    Your progress, score, and completed exercises are saved to localStorage so you can pick up exactly where you left off after closing the tab.

    📦 What's inside

    • Exercises — 27 missions across Beginner, Intermediate, and Advanced tiers with hints and XP scoring.
    • Tutorial — 8 structured lessons covering Git from first principles to advanced workflows, with diagrams and code examples.
    • Sandbox — A free-form terminal with a live commit graph and file tree. Create files, folders, branches, merge, rebase.
    • Cheat Sheet — Every essential Git command in 10 categories, instantly searchable.

    🛠️ Built with

    • Vanilla HTML, CSS, and JavaScript — zero frameworks, zero dependencies.
    • Plus Jakarta Sans for UI text · JetBrains Mono for terminal and code.
    • SVG-based commit graph renderer — drawn from scratch with dynamic lane calculation.
    • Fully responsive — works on phones, tablets, and laptops.
    • Dark & light theme with a smooth animated toggle.
